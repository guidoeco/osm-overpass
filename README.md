# osm-overpass
A set of scripts to build an OpenStreetMap Overpass docker build and server images and containers

This is this is fileset used to build OSM Overpass docker build environment and OSM Overpass server images for data analytics and consists of the docker build and management scripts plus instructions to import OSM data and run a simple set of queries against the server
